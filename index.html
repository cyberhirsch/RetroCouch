Developer Integration Guide

Complete guide for integrating ControllerManager into your games

Quick Start

// 1. Include ControllerManager in your game
<script src="ControllerManager.js"></script>

// 2. Listen for controller input events
document.addEventListener('controllerinput', function(event) {
    const input = event.detail;
    
    // Handle different input types
    if (input.type === 'button') {
        if (input.button === 'jump' && input.pressed) {
            player.jump();
        }
        if (input.button === 'fire' && input.pressed) {
            player.shoot();
        }
    }
    
    if (input.type === 'stick') {
        player.move(input.x, input.y);
    }
});

// 3. Start listening for controllers
window.ControllerManager.startListening();

ControllerManager API

Event System

controllerinput

Fired when any controller input is detected

detail.type - 'button' or 'stick'
detail.player - Player number (1-4)
detail.button - Button name (for buttons)
detail.x, detail.y - Stick position (for sticks)
controllerconnected

Fired when a controller connects

detail.player - Player number
detail.index - Gamepad index
controllerdisconnected

Fired when a controller disconnects

detail.player - Player number
detail.index - Gamepad index

Setup API

window.GameArcadeSetup.getPlayerSetup()

Get current player setup configuration

window.GameArcadeSetup.hasValidSetup()

Check if any player has valid setup

Complete Game Example

<!-- In your game HTML -->
<script src="ControllerManager.js"></script>
<script>
class ExampleGame {
    constructor() {
        this.setupEventListeners();
        ControllerManager.startListening();
        this.setupAPI();
    }
    
    setupEventListeners() {
        document.addEventListener('controllerinput', (e) => this.handleInput(e));
        document.addEventListener('controllerconnected', (e) => this.handleConnection(e));
        document.addEventListener('controllerdisconnected', (e) => this.handleDisconnection(e));
    }
    
    handleInput(event) {
        const input = event.detail;
        
        if (input.player === 1) { // Player 1 controls
            if (input.type === 'button' && input.button === 'jump' && input.pressed) {
                this.player1.jump();
            }
            if (input.type === 'stick') {
                this.player1.move(input.x, input.y);
            }
        }
    }
    
    setupAPI() {
        // Get player setup info
        const setup = window.GameArcadeSetup.getPlayerSetup();
        console.log('Current setup:', setup);
    }
}

const game = new ExampleGame();
</script>

Important Notes

Performance

ControllerManager polls at 60 FPS. Only listen to events you need to avoid performance issues.

Setup Validation

Always check if players have valid setup before starting your game. Use GameArcadeSetup.hasValidSetup().

Player Assignment

Players are assigned based on setup order. Player 1 uses first connected controller, etc.

Hot Swapping

Controllers can be connected/disconnected during gameplay. Check connection status as needed.

Troubleshooting

Controllers not responding?
  • Check if ControllerManager is loaded: console.log(window.ControllerManager)
  • Verify player setup exists: console.log(window.GameArcadeSetup.getPlayerSetup())
  • Ensure setup has valid profiles assigned
Events not firing?
  • Call ControllerManager.startListening() after adding event listeners
  • Check browser console for errors
  • Verify controllers are connected
